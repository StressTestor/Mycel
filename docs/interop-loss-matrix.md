# interop loss matrix

status: v0.1 baseline

date: 2026-05-28

v0.1 only hardens the decision surface. export code waits for later phases.

| shape | supported v0.1 fields | lost ecology features |
| --- | --- | --- |
| Mycel-native | antibodies, signatures, refusal mode, confidence, expiry, remediation, examples | none for v0.1 fail-pattern immunity |
| Hermes-compatible | name, instructions, examples, notes | antibodies are inert notes; no refusal enforcement, expiry, substrate gates, or source lineage |
| OpenClaw-compatible | plugin or skill description, examples, metadata | antibodies are metadata only; no substrate-conditioned matching or refusal enforcement |
| agentskills.io-compatible | skill name, description, examples, metadata | Mycel ecology fields degrade to inert metadata |

Exports must declare lost features instead of implying another runtime enforces
Mycel policy.
