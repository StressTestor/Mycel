# v0.3 blind-review evidence

confidence key: **solid** means verified or strongly supported. **directional** means the shape is likely right, but details may change. **vibes** means a useful hypothesis, not a fact.

this file records the out-of-band evidence for the v0.3 roadmap metric:

> at least 10 handoff specs can be reviewed and executed manually without reading the prior full transcript.

## method

the deterministic half lives in `crates/mycel-tests/tests/executable_specs_harness.rs`: it loads
`crates/mycel-tests/tests/fixtures/executable_specs.jsonl` (12 specs) and asserts each clears the
`SelfSpec::is_executable()` bar (see ADR 0014 for the exact heuristic). that proves structural
self-sufficiency, not human-judged self-sufficiency.

the qualitative half, recorded here, is a blind-reviewer pass. each of the 12 specs was handed to a
separate fresh agent with **no access to this conversation, the codebase, or any other spec**. each
reviewer was asked: given only this spec, can you name your first concrete action, and is the spec
self-sufficient (could you execute it without the originating transcript)? a reviewer that had to
guess what "done" means or where to work was instructed to answer no.

## interpretation

the single "no" (spec 3) is a feature, not a failure: the reviewer rejected it precisely because the
one API it depended on (`Substrate::runs_since`) was carried as a *directional*-confidence inherited
claim rather than a *solid* one. that is the confidence-tagging discipline doing its job — a reviewer
could see the weak link and correctly refused to act cold on it. the other 11 specs carried their
load-bearing claims at *solid* confidence with concrete source pointers, and every reviewer could
name a precise first action without the transcript.

this is consistent with the v0.3 load-bearing assumption — *manual self-specs are enough to prove
handoff quality before spawning* — at **confidence: directional**. it is not yet *solid* because the
corpus is small (12) and the reviewers judged "could I start" rather than executing to green.

## reproduction

- deterministic bar: `cargo test -p mycel-tests --test executable_specs_harness -- --nocapture`
- blind pass: hand each line of `executable_specs.jsonl` to an isolated agent with the prompt in
  this file's method section. it is intentionally not automated in CI (it needs independent agents).
