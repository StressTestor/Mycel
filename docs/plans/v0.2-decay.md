## schema: runs table (v4)

owned by `Db::run_migrations()` in `crates/mycel-core/src/lib.rs` via the `FULL_SCHEMA_SQL` constant.
`Db` is a shared connection wrapper that also owns the `audit_log` table. `AntibodyStore` (antibody
schema, user_version 3) and `Db` (runs + audit_log, user_version 4) are separate store types.

### columns

| column | type | nullable | default | semantics |
| --- | --- | --- | --- | --- |
| `id` | TEXT PK | no | - | uuid string |
| `kind` | TEXT | no | - | `observation` \| `mutation` \| `eval` |
| `status` | TEXT | no | - | `applied` \| `reverted` \| `failed` |
| `summary` | TEXT | no | - | free-text run summary |
| `confidence` | TEXT | no | - | `solid` \| `directional` \| `vibes` |
| `created_at` | INTEGER | no | - | unix timestamp (seconds), set at insert |
| `expires_at` | INTEGER | yes | NULL | ttl expiry; NULL = no ttl |
| `no_compost` | INTEGER | no | 0 | 0/1 boolean; 1 = preserve 100% |
| `decay_state` | TEXT | yes | NULL | NULL = live; `retained` \| `distilled` \| `decayed` |
| `decayed_at` | INTEGER | yes | NULL | unix ts when maintenance acted on this row |
| `distilled_summary` | TEXT | yes | NULL | compressed gist (directional records only) |

### decay behavior

| confidence | no_compost | behavior |
| --- | --- | --- |
| solid | 0 | retained (summary preserved, decay_state = 'retained') |
| directional | 0 | distilled (summary compressed to distilled_summary, decay_state = 'distilled') |
| vibes | 0 | decayed (body dropped, decay_state = 'decayed') |
| any | 1 | no-op (maintenance skips this row entirely) |

maintenance filter: `decay_state IS NULL AND expires_at <= now AND no_compost = 0`

## rust API

all in `crates/mycel-core/src/lib.rs`.

### types

- `RunKind`: `Observation | Mutation | Eval`
- `RunStatus`: `Applied | Reverted | Failed`
- `DecayState`: `Retained | Distilled | Decayed` (typed enum, maps to string column)
- `Run`: struct with all 11 columns mapped

### Db

```rust
pub struct Db { pub(crate) conn: Connection }

impl Db {
    pub fn open(path: impl AsRef<Path>) -> Result<Self>
    pub fn open_in_memory() -> Result<Self>
}
```

### Substrate

```rust
pub struct Substrate<'a> { conn: &'a Connection }

impl<'a> Substrate<'a> {
    pub fn new(db: &'a Db) -> Self
    pub fn insert(kind, status, summary, confidence) -> Result<String>
    pub fn insert_with_decay(kind, status, summary, confidence, expires_at, no_compost) -> Result<String>
    pub fn get(id: &str) -> Result<Option<Run>>
    pub fn list() -> Result<Vec<Run>>
}
```

`insert` auto-sets `created_at` from wall clock. `insert_with_decay` sets `expires_at` and
`no_compost`; `decay_state`, `decayed_at`, and `distilled_summary` are always NULL at insert.

## time injection

see ADR 0008. all NEW maintenance code takes `now: i64` as an explicit parameter. the CLI layer
passes the real clock. fixtures pass a fixed timestamp. v0.1 code is left unchanged.
